Simulation Help

The Simulation HTML knowledge object is a browser based knowledge object that displays the on-screen interactions as a series of annotated images in the form of Show, Guide, and Test simulations. It guides and helps you to understand the conceptual knowledge of a process.

How does the Show, Guide, and Test modes work?

To experience how the process is actually performed, you can simulate the process. The steps to be performed may or may not be displayed. This depends on the mode of the activity that you select from the given option and understand the process. You can use the guide that is present in the Simulation to learn the steps that are involved in the process. After you understand the process well, you can take the assessment. After you complete the assessment, the feedback regarding your performance is displayed.

The different modes are described as follows:

  • Show: This mode displays the whole application step by step guided by an audio tutor to teach you how to use the application. The steps are displayed on the cuecard and played accordingly. If the steps are not displayed, click the cuecard icon to view the steps. You can also select the steps in random from the cuecard. After you have viewed the process, the Show Mode dialog window is displayed. You can select an option to repeat the show mode, proceed to guide mode, or return to the home page.

  • Guide: This mode helps you to understand and perform the steps by yourself. If the steps are not displayed, click the cuecard icon to view the steps. Try to perform the process accordingly by viewing the instruction for each step displayed on the cuecard. If you do not perform a particular step correctly, the tutorial provides you with hints and guides you with the correct procedure for that step. When you have completed the process via the Guide mode, you can select an option to repeat the guide mode, proceed to test mode, or return to the home page.

  • Test: This mode helps you to take assess or test your conceptual knowledge of the process. To view the steps, click Show Details on the Test Data screen. Follow the instructions which are displayed on the Show Details screen and perform the process in the listed sequence. After you have completed the test, feedback is provided regarding your performance. Select an option to view the status of the steps, take the same test again, or return to the home page.
